# 今日前端技术速览

通过网络搜索获取今日前端技术动态，以简洁列表形式呈现。

<!-- Corrected: 2026-01-13 | was: 硬编码年份 "2026" | reason: 改为动态年份，避免过时 -->
## 执行流程

1. **搜索最新资讯**
   - 使用 WebSearch 工具搜索今日前端技术相关内容
   - 搜索关键词示例（使用当前年份）：
     - "前端技术 今日 <当前年份>"
     - "frontend news today <当前年份>"
     - "React Vue Angular 最新动态"
     - "JavaScript framework updates"
   - 注意：始终使用当前日期的年份，避免搜索过时内容

2. **筛选有价值内容**
   - 优先选择：新版本发布、重要更新、新工具/库
   - 过滤掉：广告、过时内容、重复信息

3. **整理输出**
   - 提取 5-10 条有价值的信息
   - 每条包含：标题 + 一句话摘要

## 输出格式

```markdown
## 今日前端技术速览 (YYYY-MM-DD)

1. **[标题]** — 一句话摘要
2. **[标题]** — 一句话摘要
3. **[标题]** — 一句话摘要
...

Sources:
- [来源标题](URL)
- [来源标题](URL)
```

## 示例输出

> 以下为格式示例，版本号仅供参考，实际输出以搜索结果为准。

- [来源标题](URL)

## 注意事项

- 始终注明信息来源（附带链接）
- 优先中文资讯，英文内容简要翻译
- 如当日无重大新闻，可扩展至本周动态

## 搜索优化技巧

### 英文搜索更有效

中文搜索结果往往是趋势总结类文章（如"2025年前端趋势"），而英文搜索能获取更及时的框架更新、版本发布等一手信息。

**建议搜索策略：**
```
中文搜索：获取国内技术社区动态
英文搜索：获取框架官方更新、国际技术新闻
```

### 并行多源搜索

单一关键词难以覆盖全面，建议并行发起多个搜索：

| 搜索方向 | 关键词示例 |
|---------|-----------|
| 综合动态 | "frontend news today <年份>" |
| 框架更新 | "React Vue Angular latest updates <年份>" |
| 中文社区 | "前端技术 最新动态 <年月>" |

并行搜索可同时获取不同维度的信息，整合后输出更全面。
